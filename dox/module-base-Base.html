Class: Base

base~ Base

The base of all classes in the system, this is one of the few pure "classes" in core of the system. It is a pretty clean little class whose primary purpose is to surface the composition chains and a basis for storing options on mixin and subclass instances. Options are handled at the instance rather than the prototype level so that multiple instances don't compete for default values.
